===================================================================
          STM32F103C8T6 引脚分配表 (智能台灯 V13.1 - DMA版)
===================================================================

[1. 通信接口 (与 ESP32 连接)]
-------------------------------------------------------------------
功能          STM32引脚      配置模式                备注
UART1_TX      PA9           复用推挽输出            DMA1_Channel4 (发送)
UART1_RX      PA10          浮空/上拉输入           DMA1_Channel5 (循环接收)
GND           GND           共地                    必须连接
-------------------------------------------------------------------
* 波特率: 115200 bps
* 协议: JSON ({"cmd":"light",...})
* 机制: DMA 循环接收 + 主循环轮询解析 (无中断频繁开销)


[2. 照明控制 (双色温 LED)]
-------------------------------------------------------------------
功能          STM32引脚      配置模式                备注
Warm PWM      PA6           复用推挽输出            TIM3_CH1 (1kHz)
Cold PWM      PA7           复用推挽输出            TIM3_CH2 (1kHz)
-------------------------------------------------------------------
* 驱动方式: MOSFET / LED 驱动芯片
* 调光深度: 0-1000 级


[3. 人机交互 (编码器 & 按键)]
-------------------------------------------------------------------
功能          STM32引脚      配置模式                备注
Encoder A     PB6           浮空/上拉输入           TIM4_CH1 (硬件编码器)
Encoder B     PB7           浮空/上拉输入           TIM4_CH2 (硬件编码器)
Mode Button   PB1           上拉输入                EXTI / 轮询 (KeyManager)
-------------------------------------------------------------------
* 编码器: 用于调节亮度/色温
* 按键: 用于切换模式 (单击/双击/长按)


[4. 显示接口 (OLED)]
-------------------------------------------------------------------
功能          STM32引脚      配置模式                备注
I2C_SCL       PB8           开漏输出                软件模拟 I2C (原 I2C1)
I2C_SDA       PB9           开漏输出                软件模拟 I2C (原 I2C1)
-------------------------------------------------------------------
* 屏幕: 0.96寸 OLED (SSD1306)
* 地址: 0x78


[5. 传感器组 (Sensor Hub)]
-------------------------------------------------------------------
功能          STM32引脚      配置模式                备注
LDR (光敏)    PA0           模拟输入                ADC1_IN0
DHT11 (温湿)  PA1           推挽输出/上拉输入       单总线协议
Gesture SCL   PB10          开漏输出                软件模拟 I2C (原 I2C2)
Gesture SDA   PB11          开漏输出                软件模拟 I2C (原 I2C2)
Gesture INT   PB5           上拉输入                (可选) 手势中断
-------------------------------------------------------------------
* PAJ7620: 手势识别传感器 (地址 0xE6)


[6. 调试接口 (SWD)]
-------------------------------------------------------------------
功能          STM32引脚      备注
SWDIO         PA13          ST-Link / DAP-Link 数据线
SWCLK         PA14          ST-Link / DAP-Link 时钟线
-------------------------------------------------------------------


[DMA 通道占用一览]
-------------------------------------------------------------------
DMA1_Channel1: ADC1 (LDR) - *如果使用了ADC DMA*
DMA1_Channel4: USART1_TX (通信发送)
DMA1_Channel5: USART1_RX (通信接收)
-------------------------------------------------------------------
