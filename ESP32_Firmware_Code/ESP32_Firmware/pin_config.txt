# 项目名称: AI 智能台灯 (ESP32-S3 大脑端)
# 硬件版本: V1.0 (基于 ESP32-S3-DevKitC-1)
# 日期: 2023-10-xx
# ==============================================================================

# ------------------------------------------------------------------------------
# [1] 系统与调试 (System & Debug)
# ------------------------------------------------------------------------------
# USB_D-            : GPIO 19  (保留，用于 USB 下载/JTAG)
# USB_D+            : GPIO 20  (保留，用于 USB 下载/JTAG)
# UART0_TX          : GPIO 43  (保留，用于 Log 查看)
# UART0_RX          : GPIO 44  (保留，用于 Log 查看)
# Onboard_RGB_LED   : GPIO 48  (板载 WS2812，用于状态指示/验证)
# Boot_Button       : GPIO 0   (系统启动引脚，仅用于下载，不建议复用)

# ------------------------------------------------------------------------------
# [2] 验证外设 (Verification)
# ------------------------------------------------------------------------------
# Test_Button       : GPIO 21  (外接杜邦线按钮，用于移植测试)
#                      -> 连接方式: 一端接 GPIO 21，一端接 GND (开启内部上拉)

# ------------------------------------------------------------------------------
# [3] 显示系统 (Display - SPI Interface)
# 设备: ILI9486 (LCD) + XPT2046 (Touch)
# 总线: SPI2 (FSPI)
# ------------------------------------------------------------------------------
# LCD_SCK           : GPIO 12  (SPI 时钟)
# LCD_MOSI          : GPIO 11  (SPI 数据写)
# LCD_MISO          : GPIO 13  (SPI 数据读 - 触摸共用)
# LCD_CS            : GPIO 10  (屏幕片选)
# LCD_DC            : GPIO 9   (数据/命令切换)
# LCD_RST           : GPIO 4   (复位)
# LCD_BL            : GPIO 5   (背光控制)
# Touch_CS          : GPIO 6   (触摸片选)
# Touch_IRQ         : GPIO 7   (触摸中断)

# ------------------------------------------------------------------------------
# [4] 听觉系统 (Audio - I2S Interface)
# 设备: INMP441 (Mic) + MAX98357A (Amp)
# 总线: I2S0
# ------------------------------------------------------------------------------
# I2S_BCLK          : GPIO 42  (位时钟 - 共用)
# I2S_LRCK          : GPIO 41  (左右声道时钟 - 共用)
# Mic_SD_IN         : GPIO 40  (麦克风数据入)
# Amp_SD_OUT        : GPIO 39  (功放数据出)

# ------------------------------------------------------------------------------
# [5] 神经连接 (Communication - UART)
# 设备: STM32F103 (小脑)
# 总线: UART1
# ------------------------------------------------------------------------------
# UART_TX_To_STM32  : GPIO 17  (连接 STM32 PA10)
# UART_RX_From_STM32: GPIO 18  (连接 STM32 PA9)

# ------------------------------------------------------------------------------
# [6] 交互输入 (Input - Encoder)
# 设备: EC11 旋转编码器
# ------------------------------------------------------------------------------
# Enc_A             : GPIO 1   (A 相)
# Enc_B             : GPIO 2   (B 相)
# Enc_Btn           : GPIO 14  (编码器自带按键)

# ==============================================================================
# 资源统计
# 总占用 GPIO: 19 个 (不含系统引脚)
# 剩余可用 GPIO: 约 8-10 个 (取决于具体模组封装)
# 潜在冲突检查: 已避开 Flash (26-32) 和 JTAG (3) 及 Strapping (45,46)
# ==============================================================================
